AI-Powered Desk Lamp  ·  v2  ·  Full Stack: Laptop + ESP32

Software Overview

What does it do?

Luxo is an AI-powered physical lamp. You speak to it, it listens, thinks, replies out loud, and physically moves its head, neck, and joints to express emotions. It also lights up with different colors depending on how it's feeling. You can also wave your hand in front of a camera and the lamp will follow it around.

Conversation Mode

You talk → Luxo transcribes your voice, sends it to Gemini AI, speaks the reply out loud, and moves to match the emotion of its answer.

Hand Tracking Mode

Say "follow me" → the lamp locks onto your hand using the camera and follows it in real-time. Say "talk to me" to go back to conversation.

Platforms & Technologies

🖥️

Laptop (Python)

Runs the main brain of the project.

  • Flask web server
  • Whisper — speech-to-text
  • Gemini AI — generates replies
  • Edge TTS — text-to-speech
  • MediaPipe — hand detection
  • Bleak — Bluetooth (BLE) to ESP32
⚡

ESP32 (MicroPython)

Controls all hardware on the lamp itself.

  • BLE (Bluetooth) receiver
  • 4× Servo motors
  • NeoPixel LED ring (12 LEDs)
  • Idle animations when not in use
  • JSON command protocol
🌐

Browser (HTML/JS)

The user interface — runs on your laptop's browser.

  • Records your voice via microphone
  • Sends audio to Flask server
  • Shows Luxo's reply as text
📡

ESP32-CAM (optional)

A second ESP32 with a camera — used for hand tracking mode.

  • Streams video over WiFi
  • MediaPipe reads the stream
  • Replaces laptop webcam if set up

System Logic — Step by Step

Here is exactly what happens from the moment you speak to when the lamp moves.

Conversation Mode
1

You speak into the browser

The browser records your voice using the microphone and sends the audio file to the Flask server running on your laptop at localhost:5050.

2

NeoPixel flashes white — "I heard you"

Immediately, flash_ack() is called. The NeoPixel ring flashes white for 120ms to let you know Luxo received the audio before processing begins.

3

Whisper converts your voice to text

The faster-whisper model (tiny, running on CPU) processes the audio file and returns a text transcript of what you said.

4

Mode switch check

The text is checked for special phrases. "Follow me" → switches to hand tracking mode. "Talk to me" → returns to conversation. Otherwise continues to AI.

5

Lamp goes to "thinking" mode

While waiting for Gemini, the lamp tilts its head and the NeoPixel shows a blue chasing-dot animation — visually showing that it's thinking.

6

Gemini AI generates a response

The transcript is sent to gemini-2.5-flash. The AI always starts its reply with an emotion tag like [EXCITED], [HAPPY], [THINK], etc., followed by 1–3 sentences.

7

Emotion parser reads the tag

The EmotionParser reads the tag and maps it to a servo behavior name — for example [EXCITED] → wiggle, [AGREE] → nod. It strips the tag from the spoken text.

8

Three things happen at once

(a) TTS speaks the reply text through the laptop speaker.
(b) The behavior command is sent over Bluetooth to the ESP32 — the lamp physically moves.
(c) The NeoPixel changes color to match the emotion.

9

ESP32 receives and executes

The ESP32's BLE receiver gets the JSON command, parses it, and runs the matching servo behavior (e.g. nod, wiggle, curious tilt). The NeoPixel animation also updates simultaneously.

Hand Tracking Mode
1

You say "follow me"

The voice command is detected and tracker.enable() is called. The NeoPixel switches to rainbow animation to show tracking is active.

2

Camera captures frames

A background thread opens the ESP32-CAM stream (or laptop webcam as fallback) and reads frames at ~20fps.

3

MediaPipe detects your hand

Each frame is processed by MediaPipe Hands, which returns the X/Y position of the hand center and the hand's bounding box size (used to estimate distance).

4

Smooth servo commands sent to lamp

Hand X → pan angle (left/right). Hand Y → tilt angle (up/down). Hand size → lean angle (forward/back). Values are smoothed and only sent when they change by 1° or more to avoid jitter.

File Reference

File Runs on What it does
esp32_main.py ESP32 Main firmware. Runs on the lamp's ESP32 chip. Sets up BLE, receives JSON commands, drives servos and NeoPixel, plays idle animations.
app.py Laptop Flask web server. The central hub — receives browser audio, runs Whisper, calls Gemini, calls TTS, and sends commands to the lamp.
ai_brain.py Laptop Connects to Gemini AI. Sends user text, gets back a reply with an emotion tag. Keeps conversation history.
emotion_parser.py Laptop Reads the AI's emotion tag (e.g. [EXCITED]) and maps it to a servo behavior name. Strips the tag from the text so TTS doesn't say it aloud.
lamp_link.py Laptop Handles the Bluetooth (BLE) connection from laptop to ESP32. Also supports USB serial. Sends JSON commands to the hardware.
neopixel_controller.py Laptop Manages LED lighting commands. Maps each emotion to a color and animation effect, then sends the command via lamp_link to the ESP32.
hand_tracker.py Laptop Background thread. Opens camera stream, runs MediaPipe to detect hand position, and sends real-time pan/tilt/lean servo commands.
tts.py Laptop Converts Luxo's text reply to speech using Edge TTS (Microsoft voices) and plays it through the laptop speaker using pygame.
test_lamp_movements.py Laptop Test script. Cycles through all servo behaviors one by one — useful for checking the hardware works without running the full AI system.

NeoPixel LED States

Each emotion the AI expresses is matched to a unique LED color and animation effect on the 12-pixel NeoPixel ring.

HAPPY

Warm yellow · pulse

EXCITED

Cyan · fast strobe

CURIOUS

Violet · breathe

THINK

Deep blue · chase dot

AGREE

Green · 2× blink

DISAGREE

Amber-red · flicker

IDLE

Warm white · slow breathe

HAND TRACK

Rainbow · spinning

THINKING

Blue · chasing dot

How it works: The laptop side (neopixel_controller.py) sends a JSON command like {"command":"neo_set","r":255,"g":200,"b":30,"effect":"pulse","speed":80,"brightness":200} over Bluetooth to the ESP32. The ESP32 firmware (esp32_main.py) receives this and drives the NeoPixel ring with the correct color and animation in its main loop.

Laptop ↔ ESP32 Communication

Protocol

The laptop sends plain JSON text lines to the ESP32 over Bluetooth Low Energy (BLE) — or USB serial as a fallback. Each command ends with a newline character so the ESP32 knows when the message is complete.

Servo behavior command

{"command":"behavior",
"name":"wiggle"}

Triggers a named servo animation on the ESP32.

Direct servo move

{"command":"move",
"joint":"pan","angle":110}

Moves one joint to an exact angle (0–180°). Used by hand tracker.

NeoPixel set

{"command":"neo_set","r":160,
"g":50,"b":255,"effect":"breathe",
"speed":60,"brightness":180}

Sets LED color + animation effect.

NeoPixel flash

{"command":"neo_flash",
"r":255,"g":255,"b":255,
"duration_ms":120}

One-shot flash for a set duration, then restores previous state.
